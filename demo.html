üí° What is a UTXO?

UTXO = Unspent Transaction Output
Each Bitcoin transaction can have multiple outputs (like giving change). vout is the output number (0, 1, 2, etc.)

Example: If a transaction sends to 2 addresses:
- vout 0 = First recipient
- vout 1 = Second recipient (often "change" back to sender)

Example: abcd1234...ef5678 (64 characters)
Usually 0 or 1. Check on block explorer if unsure.
